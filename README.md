# VMKMilk Demo Store

Визуальный демо-магазин без серверной логики и без регистрации.

## Что есть
- каталог на главной (20 товаров)
- поиск, фильтр по типу, сортировка
- интерактивная корзина (добавление/удаление/количество)
- демо-оформление заказа и история заказов
- адаптивный интерфейс с анимациями

## Локальный запуск

```bash
python3 -m http.server 4173
```

Откройте `http://127.0.0.1:4173`.

## Деплой

Проект без сборки — подходит для GitHub Pages (`root`).
