# План разработки визуального демо интернет-магазина VMKMilk

> Цель: сделать приятный, адаптивный, полностью клиентский демо-магазин **без серверной логики и без регистрации**.
> Все интеракции работают визуально (по JavaScript в браузере), чтобы быстро согласовать UX/UI.

## Статус текущей версии

### ✅ Готово сейчас
- Главная страница-магазин с 20 товарами из каталога VMKMilk.
- Поиск по каталогу (регистр + базовая устойчивость к опечаткам).
- Фильтр по категориям и сортировка по цене/названию/популярности.
- Интерактивная корзина: добавление, изменение количества, подсчёт суммы.
- Демо-оформление заказа: модальное окно успеха + запись в демо-историю покупок.
- Блок доступных дат доставки в формате UI-демо.
- Адаптивный интерфейс + плавные анимации карточек/кнопок.
- Структура подходит для деплоя на GitHub Pages без сборки.

### 🔜 Следующие шаги (только визуал)

## Нефункциональные принципы демо
- **Производительность:** минимум зависимостей, статический HTML/CSS/JS, lazy-loading изображений.
- **Адаптивность:** mobile-first и удобство на экранах от 320px.
- **Понятность:** явные подписи «демо» там, где нет реальной бизнес-логики.
- **Расширяемость:** простая структура, чтобы легко перейти к интеграции backend/1C.
